每次回應結尾，改用**數字選項**取代文字問句，讓使用者直接回數字即可繼續。

**Why:** 使用者不喜歡重複輸入文字，數字選單更有效率。

**How to apply:** 所有回應的「想繼續討論...」區塊，一律改為：
```
1. xxx
2. xxx
3. xxx
```
使用者回「1」就繼續討論第1項。
